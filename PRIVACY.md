# Privacy Policy

## Data Collection

GitHub Better CSV Diff does **not** collect, store, or transmit any user data.

## How It Works

This extension reads the contents of CSV diff elements on GitHub pages (Pull Requests and commit diffs) and renders them as side-by-side tables directly in your browser. All processing happens locally — no data is sent to any external server.

## Permissions

- **Host permission (`github.com`)**: Required to detect CSV diffs and inject the table view overlay on GitHub pages. No other sites are accessed.

## Contact

If you have any questions, please open an issue at https://github.com/letconst/github-better-csv-diff/issues.
